About

2 min · 411 words · Ömer

I’m a computer enthusiast, scener and an old MMORPG gamer currently living in northern part of the world. I drink copious amounts of coffee☕️, read a lot of books📖, listen rock music🎸, ßeta test some apps🖥 and spend a lot of time in front of my computer and vintage old ones.

After internet technologies evolved, I started to create web pages coded in PHP that generates HTML and should run smoothly on any browser. I also create examples of web sites that use Python and many successful, high-performance, high-scale web sites written in Java, PHP, Ruby, .NET, Perl, Javascript, and just about anything else you can think of. Nowadays i create minimalistic valid xhtml web pages with little additon css.

A list (possibly outdated) of facts about me…

  • I am interested in beer. Both drinking and brewing techniques.
  • I am a supporter of VHEMT.
  • I favor Perl, Ruby, Objective-C, Go, Javascript & Lua.
  • I consume a lot of media. TV-series, Movies, Audio, Books, Computer & PlayStation®4 Games and more.

Some projects I am involved in…

  • Extended DOS Collection – Easy to use PC DOS command line programs written entirely in x86 Assembly code from 1993.
  • Commodore 64 Demos – A WebSite online in early 2000s. About Commodore 64 Demos & techie stuff.
  • HVGC – High Voltage Graphic Collection, WebSite about Commodore 64 art. Never published.

Bilgisayarlar ile tanıştığım 1987 yılından beri hep bilgisayar dünyasının içinde oldum. Şimdilerde eski nostaljik bilgisayarlar ile ilgileniyorum, biraz oyun oynuyorum ve kendimi oyalamak amacıyla 2006 yılında başlattığım bu blog’a halen birşeyler yazmaya devam ediyorum.

Blog yazıları yazmaya başladığım dönemlerde internet teknolojileri ile ilgilenmeye başladım ve PHP ile minimalist web siteleri tasarladım. Java, Ruby, Python ve Perl ile ilgilensem de bunlar hep hobi düzeyinde kaldı.

Bugünlerde MMORPG oyunları oynuyorum, biraz kahve☕️ içiyorum, fırsat bulursam kitap-dergi📖 okurum, inanmayacaksınız ama hala rock-metal-death-grindcore🎸 müzik dinliyorum. ßeta test🖥 yapmayı ve sistemleri kurcalamayı hala çok seviyorum.
